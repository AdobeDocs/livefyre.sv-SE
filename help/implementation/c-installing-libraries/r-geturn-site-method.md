# getUrn-webbplatsmetod{#geturn-site-method}

Den här metoden returnerar URN för den här platsen.

## Java-exempel {#section_nyl_ycs_rz}

Användning:

```
site.getUrn();
```

Exempelutdata:

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```
